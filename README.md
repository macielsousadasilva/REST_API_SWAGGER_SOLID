
  

## REST_API_SWAGGER

  

### Setup

  

  

```bash

  

npm install

  

```

  

  

### Iniciar como desenvolvimento com nodemon e tsc --watch

  

  

```bash

  

npm run dev

  

```

  

  

Após iniciar acesse: `http://localhost:3000/cats`

  

  

### Iniciar sem o nodemon e tsc --watch

  

  

```bash

  

npm start

  

```

  

  

Após iniciar acesse: `http://localhost:3000/cats`
